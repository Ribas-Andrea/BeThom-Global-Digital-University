# README - Formation HTML / CSS Responsive

## 📚 Présentation

Ce projet retrace l’ensemble de la formation réalisée autour de la création d’un site web moderne en **HTML5** et **CSS3**, avec une approche progressive allant de la structure de base jusqu’au responsive design complet.

L’objectif principal de cette formation était de :

- Comprendre la structure d’une page web
- Créer une interface moderne et responsive
- Maîtriser le positionnement CSS
- Construire différentes sections d’un portfolio/site vitrine
- Adapter le site à tous les formats d’écran

---

# 🛠️ Technologies utilisées

- HTML5
- CSS3
- Flexbox
- Responsive Design
- Media Queries

---
